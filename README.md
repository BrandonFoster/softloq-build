# softloq-build
Python console app for building C++ apps, libs, and tests

# install
`> python -m pip install -e .`

# uninstall
`> python -m pip uninstall softloq-build`

# usage
```
> softloq-build
Hello World!
```

# running tests
pytest must be installed: `> pip install pytest`

run pytest in the project directory: `> pytest`
